## 上传规则

文件名的extension必须以下之一

* `JPG`
* `jpg`
* `png`
* `PNG`
* `gif`
* `GIF`
* `txt`: 若没有图的报单,使用文本文件, 文件名是报单的数据

文件名格式: `时间#序号#资金金额#仓位`

增加资金: `时间#序号#资金金额#仓位#+数字`
减少资金: `时间#序号#资金金额#仓位#-数字`

融资账户: 

* `时间#序号#资金金额#仓位#e1`: 交割单
* `时间#序号#资金金额#仓位#e2`: 资金账户数据
